UnSeenCat

by Iwan Gabovitch @qubodup

Hunt some alert mice in this mini mouse hunting game! Thief, Portal and Metal Gear tropes/references included!

Start

  1. Install the Free, open source, Linux/Mac OS X/Windows compatible LÖVE.
  2. Unpack unseencat.zip.
  3. Drag-and-drop the folder on the LÖVE icon.
